# **App Name**: KOVentures Hub

## Core Features:

- Company Overview: Showcase KOVentures LTD as an umbrella company.
- Renewable Energy Solutions: Detail offerings in clean, renewable energy for residential, agricultural, and commercial businesses.
- Calibration Bench Solutions: Present solutions for pressure and temperature calibration benches.
- Other Ventures Section: A section to present other upcoming ventures of the company.
- Contact Form: Contact form that users can use to send their requests (client-side Formspree submission).

## Style Guidelines:

- Primary color: Metallic blue family with high-contrast accents for a premium, modern look.
- Background: Light, clean surfaces with subtle gradients/glassmorphism.
- Accent color: Cool cyan/teal highlights to signal technology and clarity.
- Font pairing: 'Outfit' for headlines and 'Inter' for body text.
- Use clean, professional icons related to renewable energy, calibration, and general business concepts.
- A clean, organized layout that's easy to navigate. Highlight key information about KOVentures' various ventures.
- Subtle animations and transitions to enhance user experience and provide a sense of modernity and innovation.

## Technical Constraints

- Built as a static export (no server actions or route handlers).
- Deployed to GitHub Pages via GitHub Actions.
